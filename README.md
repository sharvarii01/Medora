# MedoraQueue - Enterprise Healthcare Ecosystem

MedoraQueue is a modern, real-time healthcare management web application designed to streamline patient queuing, doctor appointments, and hospital administration. Built with a premium, responsive UI, Medora allows seamless interaction between patients, doctors, and administrative staff.

## 🚀 Features

- **Multi-Role Authentication**: Secure login portal simulating access for Patients, Doctors, and Admins.
- **Real-Time Live Queue**: A public-facing live board that updates instantly as doctors process their patients.
- **Smart Patient Dashboard**: Patients can book appointments, track their estimated wait times, receive urgent notifications when their turn approaches, and manage their health goals.
- **Medora AI Chatbot**: An integrated virtual assistant that understands basic symptoms and provides immediate triage guidance.
- **Doctor Command Center**: Doctors can manage their daily queue, mark patients as completed, inject emergency cases, write prescriptions, and toggle their availability status.
- **Admin Analytics**: A powerful admin panel featuring Recharts data visualizations, a real-time system audit log, and revenue tracking.
- **Premium UI/UX**: Built with modern CSS featuring glassmorphism, smooth micro-animations, a Global Search command palette (Ctrl+K), and full Dark Mode support.

## 💻 Tech Stack

- **Frontend**: React (Vite)
- **Routing**: React Router DOM
- **State Management**: Zustand (with local storage persistence)
- **Styling**: Modern Vanilla CSS
- **Icons**: Lucide React
- **Charts**: Recharts
- **Notifications**: React Hot Toast
- **Animations**: Canvas Confetti

## 🛠️ Getting Started

To run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Medora.git
   cd Medora
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
